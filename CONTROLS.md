# Controls

## Driving (keyboard)

| Key | Action |
|-----|--------|
| W | Forward |
| S | Backward |
| A | Strafe left |
| D | Strafe right |
| Q | Rotate left |
| E | Rotate right |
| R | Speed up |
| F | Speed down |
| Z | Quit teleop |

Three speed levels: slow, medium, fast. Starts at slow.

## Arm (leader arm)

Move the leader arm physically. The follower arm mirrors it.

## Recording (record.py)

| Key | Action |
|-----|--------|
| Right arrow | Skip to next episode / end reset early |
| Left arrow | Discard episode and re-record |
| Escape | Stop session, encode videos, push to Hub |
